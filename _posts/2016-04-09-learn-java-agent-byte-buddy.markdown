提供一个参数， 然后把这个.class文件生成到指定的文件目录， 比如tomcat里面的webapps/ROOT/WEB-INF/classes/目录下面， 这个
方法没啥问题， 但是这个有点不方便嘛， 还要用户来提供这个目录， 这个不是把自己的实现困难的负担转嫁给用户吗（一位同事说的真理）
